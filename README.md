# HR-Analytics-Dashboard
Analyzed employee attrition using Python (Pandas, Matplotlib), uncovering key factors such as overtime, salary, job satisfaction, and department-wise turnover. Built as an end-to-end HR analytics project for data-driven workforce insights.

This project analyzes employee attrition data to identify the key factors influencing employee turnover. Using Python and Pandas, the dataset was cleaned, explored, and analyzed to generate actionable business insights for HR decision-making.

# Key focus areas include:
- Attrition Rate Analysis
- Department-wise Attrition
- Overtime Impact
- Salary vs Attrition
- Job Satisfaction Analysis
- Work-Life Balance Analysis
- Employee Retention Insights

  ## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Business Analytics
